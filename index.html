Manual Installation Instructions

To install this Progressive Web App (PWA) manually, follow the steps below depending on your device and browser:

On Desktop:

  • Google Chrome: Visit the website and click the "Add to Home Screen" prompt that appears in the address bar. Alternatively, click the three dots (menu) in the top-right corner and select "Install [App Name]".
  • Microsoft Edge: If you are using Edge, you should see an "Install" prompt in the address bar. Alternatively, click the three dots (menu) in the top-right corner and select "Apps" â†’ "Install this site as an app".
  • Mozilla Firefox: Currently, Firefox does not support PWA installation prompts, but you can manually create a shortcut to the app by saving the webpage to your desktop. It will not be installed as a full PWA with offline support, though.

On Mobile:

  • Google Chrome on Android: Open the website and you should see the "Add to Home Screen" prompt. Tap it to add the app to your home screen. If the prompt doesn't appear, tap the three dots (menu) and select "Add to Home Screen".
  • Safari on iOS: Open the website and tap the "Share" button (the square with an arrow). Scroll down and tap "Add to Home Screen".
  • Microsoft Edge on Android: Similar to Chrome, tap the three dots (menu) and select "Add to Home Screen".

Important Notes

The service worker is optional. While it enables offline functionality and caching for faster loading, the key feature that allows the app to be installed is the manifest.json file, which should be placed in the public folder. You can still install and use the app even if the service worker is not present.

Additionally, the manifest.json file must be linked in the <head> section of your HTML file using the <link rel="manifest" href="public/manifest.json"> tag. This is essential for the app to be recognized as a Progressive Web App and for the installation prompt to appear.
